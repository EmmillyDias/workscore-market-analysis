# 📊 WorkScore — Análise de Validação de Mercado

> *"Know before you go."*

## 🎯 Sobre o Projeto

O WorkScore é uma plataforma de score de risco para voluntários, criada para resolver
um problema real: avaliações desonestas e filtradas pelo medo de retaliação nas
plataformas de voluntariado.

Este repositório documenta a análise de validação de mercado conduzida em abril de 2026,
com 30 respondentes de 14 países em 3 idiomas (PT, ES, EN).

**Meu papel no projeto:** Co-fundadora responsável pela área de dados — conduzi a
pesquisa, análise quantitativa e qualitativa das respostas e geração de insights
estratégicos para a decisão de avançar para o MVP.

## 📈 Resultados da Validação

| Métrica | Resultado |
|--------|-----------|
| Respostas totais | 30 |
| Países representados | 14 |

| Nota média de experiência | 8,3 / 10 |

## 🔧 Etapas da Análise

1. **Coleta** — formulário em 3 idiomas distribuído em redes sociais e grupos
2. **Tratamento** — exportação para Google Sheets, padronização dos dados
3. **Análise quantitativa** — intenção de uso, distribuição por idioma, confiança no anonimato
4. **Análise qualitativa** — identificação de padrões nas respostas abertas em 3 idiomas
5. **Geração de insights** — validação do plano de negócio e recomendações estratégicas

## 💡 Principais Insights

- **90%** de intenção de uso positiva — resultado forte para fase de validação
- **53%** já encontrou avaliações desonestas nas plataformas — problema confirmado
- **50/50** na confiança no anonimato — proposta de valor ajustada para "anônimo E verificado"
- Assédio e segurança emergem como dimensão crítica, especialmente no público latinoamericano
